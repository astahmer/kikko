# Intro

Trong ORM allows you to run reactive SQLite queries on any platforms with any framework/lib. For the web apps it uses absurd-sql, for electron/react-native/ionic ‚Äî native sqlite calls. It is actually lib-agnostic, so you can integrate it to any framework/render libs you want.

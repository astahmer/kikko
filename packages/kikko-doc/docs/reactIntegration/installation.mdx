# Installation

Please, checkout [backend section](/backends/web) in the doc to install and configure Kikko for particular SQLite backend.

You also need to install additional package to add hooks support:

<Tabs>
  <TabItem value="yarn" label="yarn" default>

```bash
yarn add @kikko-land/react
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npm i -S @kikko-land/react
```

  </TabItem>
</Tabs>
